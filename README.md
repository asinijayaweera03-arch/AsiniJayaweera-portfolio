## Getting Started

Clone the repo and install dependencies:

```bash
git clone https://github.com/asinijayaweera03-arch/AsiniJayaweera-portfolio.git
cd AsiniJayaweera-portfolio
npm install
```

Run the dev server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

Lint the project:

```bash
npm run lint
```
